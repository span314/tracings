This webapp requires Javascript and a modern HTML5 browser. Supported browsers include the latest versions of MS Edge, Firefox, Chrome, and IE11 (no audio).

Ice Diagrams

This version (0.2.0) is a non-indexed snapshot of some work in progress. The latest version can be found here.

Diagrams are adapted from the USFSA rulebook.

FAQ

  • Can it play dance music?

    This is a common feature request I've been getting, and something I'm considering for a future version. I would need to work out synchronizing the diagrams, copyright issues with the music, and hosting.

  • What about other patterns?

    I have plans to add international dances and moves patterns in future versions. I need to work out label crowding issues with international dances as well as the fixed steps of the Finnstep.

Known Issues

  • Some of the labels may overlap. The labels are automatically placed next to the tracing lines, and the algorithm does not check for overlaps.
  • Metronome sound is a bit choppy in Firefox, and the WebAudio API is not supported by IE11.
